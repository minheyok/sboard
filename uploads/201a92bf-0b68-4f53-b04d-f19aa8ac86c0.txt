리눅스 개요 (Linux)

리눅스 -> 여러 사용자가 동시에 시스템에 로그인하고 여러 작업을 동시에
실행할 수 있는 다중 사용자 및 다중 작업 환경 제공

리눅스는 서버 개발환경 클라우드 컴퓨팅 모바일 기기 임베디드 시스템
영역에서 널리 사용


2. 리눅스 구조

커널-> HW 장치의 지원정보 및 HW를 제어하는 코드 등의 포함된 리눅스
핵심 코드

쉘 -> 사용자가 입력한 명령을 해석해 커널로 전달하는 명령어 해석기

응용 프로그램-> 리눅스 운영체제를 운영관리하기 위한 각종 프로그램

3. 리눅스 배포판

여러 단체 및 기업에서 커널과 쉘 응용프로그램들이 통합된 배포판 제작

리눅스는 Redhat 계열 Debian 계열 Slackware 계열로 나뉨

가장 대표적인 리눅스 배포판으로 CentOS, Ubuntu, Fedora


4. 가상머신

가상머신 -> Host OS 에 가상 컴퓨터를 만들어 또 다른 운영체제를 설치할수 있는 SW

PC에 설치된 OS를 Host
