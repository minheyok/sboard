리눅스

1.셀 개요

셀 -> 사용자가 입력한 명령을 해석해 Kernel로 전달하는 명령어 해석기
리눅스 터미널 환경

배시셀 -> 리눅스의 기본 셀로 자공 셀은 명령어 해석기능,프로그래밍기능
사용자 환경 설정 기능을 제공

환경변수-> 셀에서 전역으로 참조하는 전역 환경변수 현재 사용자 셀
환경에서 만 참조하는 셀 환경 변수

2. 셀 환경 설정 파일

셀 환경 설정 파일-> 사용자가 시스템에 로그인을 하면 셀 환경 설정을 위해
자동으로 실행되는 파일

셀 환경 설정 파일은 시스템 전체의 초기화를 위한 파일과 개인 유저 환경을
위한 초기화 파일 존재

3. 셀 스크립트

셀 스크립트-> 간단한 프로그래밍 언어로 기본적인 변수 반복문 제어문 지원
 별도로 compile 과정 없이 텍스트 파일 형태로 바로 실행
셀 스크립트 실행 명령어-> sh 실행 권한을 갖고 있으면 파일명으로 실행
