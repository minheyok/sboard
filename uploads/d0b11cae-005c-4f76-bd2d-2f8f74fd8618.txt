Spring Data JPA
 
1.JPA개요

JPA -> 자바 ORM 기술에 대한 API 표준으로 Java를 이용해서 데이터
유지관리 할 수 있는 기술

Hibernate -> JPA 인터페이스를 구현한 구현체이자 자바용 ORM 프레임워크

JPA는 복잡한 SQL 실행이 어렵다는 것과 JPA 기술에 애플리케이션 개발에
제대로 활용하기 위해서는 많은 SQL 개발 경험 필요함
